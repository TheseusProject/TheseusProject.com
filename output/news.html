  • Whats all the fuss about

    29 Oct, 2013

    Welcome to the project, so your likely to want to know what this is all about.

    I suppose I should give you an overview of what Theseus is. Theseus is a screen/document scraping/processing engine that is designed to allow a user to write applications in a .net language or scripts in JavaScript to download, webpages (those pages could well be behind a few different page requests and signin forms etc), JSON APIs, Excel/CSV1 documents.

    Part 1.. the core.

    Firstly there is the open source (or soon to be) Theseus engine.

    The Theseus Engine is made up of 3 main components, 1. The core engine, this is responsible for this is made up of the core engine which will be accessasble as a .net library, then we have the JavaScript/CoffeeScript (and hopefully others in the future) runtime this lets you use a CommonJS stlye module to invoke the core engine and return JSON results, then finally we have the CLI which is a wrapper around the JavaScript runtime which lets you point the CLI at a folder of scripts and run them.

    Part 2.. the platform

    Secondly we are planning a cloud hosting platform for your Theseus scripts, the platform will support all the features of the command line app so you will be easly be able to develop and test your scripts locally before uploading them to be hosted in the cloud.

    The platform will have the option for you to share your scripts for anyone to use or copy... or for a fee we will support private repositories, for those more sensative scripts.
